# Тестовое задание - Лендинг "Петербург сквозь года"

Адаптивная верстка лендинга курса акварельной живописи с Владимиром Заруцким.

## Технологии

- **Bootstrap 4.6.2** - сетка, компоненты (navbar, collapse, accordion)
- **jQuery 3.6.0** - динамические элементы
- **CSS3** - кастомные стили, анимации, clamp() для резиновой верстки
- **Font Awesome 5** - иконки социальных сетей

## Шрифты

- **BernierDistressed** - заголовки (локальный файл)
- **PT Sans** - основной текст (Google Fonts)

## Реализованный функционал

- Фиксированное меню (sticky header)
- Плавная прокрутка по якорным ссылкам
- Адаптивное бургер-меню на мобильных устройствах
- FAQ аккордеон
- Переключатель валют в тарифах (₽, ₴, $, €)
- Анимация элементов при скролле
- Подсветка активного пункта меню

## Адаптивность

Резиновая верстка с использованием:
- Bootstrap grid system (col-lg, col-md, col-sm)
- CSS clamp() для плавного масштабирования
- Media queries для 5 брейкпоинтов:
  - 1200px+ (desktop)
  - 992-1199px (laptop)
  - 768-991px (tablet landscape)
  - 576-767px (tablet portrait)
  - до 576px (mobile)
